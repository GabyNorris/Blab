# 💬 Blab — Real-Time Chat Application

> **Blab** is a personalized, full-stack, real-time messaging application built using the MERN stack and WebSockets. Originally started as a Udemy certification project, this repository showcases a fully customized, production-ready chat interface designed to be casual, punchy, and fast.

---

## 🚀 Features

* **Real-Time Messaging:** Instant, bi-directional communication powered by Socket.io (no refreshing required).
* **User Authentication:** Secure user registration, login, and session management.
* **Dynamic Chat Rooms:** Ability to create, join, and chat in different rooms or direct messages.
* **Theme Selection:** 32 different themes to choose from.
* **Responsive UI:** A clean, modern interface that looks great on both desktop and mobile screens.
* **Online/Offline Status:** Live tracking of which users are currently active and "blabbing."

---

## 🎨 Beyond the Course (Personalizations)

While the foundation of this application was built following a comprehensive Udemy certification course, I stepped away from the standard tutorial constraints to make **Blab** my own:

* **Custom Rebranding:** Replaced the generic "Chatty" tutorial concept with **Blab**—a modern, casual, and highly brandable identity.
* **UI/UX Copy Overhaul:** Refactored standard button texts and alerts to match the brand voice (e.g., changing "Send Message" to "Start Blabbing").
* **Codebase Personalization:** Custom-tailored the application state, variable naming conventions, and UI styling to reflect a unique product rather than a copy-pasted tutorial.

## 🛠️ Tech Stack

* **Frontend:** React.js, Context API / Redux (for state management), CSS/Tailwind
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (with Mongoose ODM)
* **Real-Time Protocol:** Socket.io

## 📦 Installation & Setup

To run this project locally, follow these steps:

## How to run Psychedelic Pixels

#### Step 1:

Clone the repo:
```
https://github.com/GabyNor05/Psychpix.git
```

#### Step 2:

Open Blab backend in terminal & install dependencies:

```
cd backend
npm i
```

#### Step 3:

Run Backend:

```
npm start
```

#### Step 4:

Open Blab frontend in a new terminal & install dependencies:
```
cd frontend
```

#### Step 5:

Run Frontend:

```
npm run dev
```
